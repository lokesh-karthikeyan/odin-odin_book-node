# The Odin Project - Odin-Book

## Echo

Echo is a full-stack social media platform designed for real-time interaction and community building.

## 🚀 Features

- **Authentication:** Secure JWT-based authentication with registration and login flows.
- **Social Graph:** Follow and unfollow users to curate your feed.
- **Content:** Create posts, like content, and engage through comments.
- **Messaging:** Real-time private messaging system with conversation threads.
- **Profiles:** Customizable user profiles including avatars and personal bios.
- **Media:** Support for image uploads and static asset management.

## 🛠️ Tech Stack

- **Frontend:** Bun, Fastify / Node.js,
- **Backend:** SvelteKit
- **Database:** PostgreSQL, Drizzle ORM
- **Auth:** JSON Web Tokens (JWT)
